This is James

This is James

Summary

Survive a day as James - The mildly functioning caffeine addict. James is living an uneventful life, filled to the brim with that everyday grind, and a dream that fades away for each passing day. Then, one day somebody decided to interfere.

Game created for the Game Project course at Aalto University. This is an interactive visual novel in which the player has the power to affect how the story turns out.

Responsibilities: Narrative, ink scripting, voice directing.

Tools: Unity, Ink.

WebGL Build: paledomain.itch.io/this-is-james.

Github: github.com/AaltoGAME2021/thisisjames.
